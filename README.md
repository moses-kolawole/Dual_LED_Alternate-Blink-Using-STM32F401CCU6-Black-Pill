# Dual_LED_Alternate-Blink-Using-STM32F401CCU6-Black-Pill
This project demonstrates how to control multiple GPIO output pins on the STM32F401CCU6 Black Pill Development Board using bare-metal (register-level) programming without relying on STM32 HAL libraries.  Two external LEDs are connected to separate GPIO pins and are programmed to blink alternately, producing a police-light effect.
